Event Webpage

A modern and responsive event landing page built using HTML, CSS, and JavaScript.This webpage displays event information such as the title, description, schedule, speakers, and a registration section.
Designed with an aesthetic and professional color palette.

Features:
  Responsive event landing page layout
  Modern aesthetic UI
  Event details section
  Schedule or agenda section
  Speaker cards (if included)
  Registration or call-to-action button
  Fully built using HTML, CSS, and JavaScript

Technologies Used:

HTML
CSS
JavaScript

How to Use:

Clone or download the project
Open the project folder
Run the index.html file in a web browser
View all event details directly on the webpage

Project Structure:

index.html – main webpage layout

style.css – styling and UI design

script.js – interactive functionality (if any)
